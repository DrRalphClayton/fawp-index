# Screenshots

## Adding screenshots to the README

1. Run a scan on `SPY, QQQ, GLD, BTC-USD` at 2y period
2. Take a screenshot of the Scanner tab showing flagged assets
3. Save as `scanner_tab.png` in this folder
4. The README will automatically reference it
